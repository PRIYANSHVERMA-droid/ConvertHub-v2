## 🚀 Overview

**ConvertHub v2** is a modern desktop file converter focused on clean UX and practical formats — not overwhelming users with unnecessary options.

It integrates powerful engines:

- **FFmpeg** → audio & video  
- **LibreOffice** → document conversion  
- **7-Zip** → archive handling  

All bundled directly — no setup required.
